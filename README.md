**Barbearia API**

API REST em desenvolvimento para gerenciamento de uma barbearia.
O projeto foi construído com foco em boas práticas, utilizando Node.js, Express, Sequelize e PostgreSQL rodando dentro de containers Docker.

🚀 Tecnologias Utilizadas

Node.js

Express

Sequelize

PostgreSQL

Docker

📌 Funcionalidades (em desenvolvimento)

Cadastro de clientes

Agendamento de horários

Cadastro de serviços (ex: corte, barba, hidratação)

Controle de barbeiros

Histórico de agendamentos
